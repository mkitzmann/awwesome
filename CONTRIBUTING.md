# Contributing to Awwesome

Thanks for your interest in contributing! Here are some ways you can help.

## Reporting Bugs

Found a bug? Please [open a bug report](https://github.com/mkitzmann/awwesome/issues/new?template=bug_report.yml) using the issue template.

## Suggesting Features

Have an idea? Please [open a feature request](https://github.com/mkitzmann/awwesome/issues/new?template=feature_request.yml) using the issue template.

## Pull Requests

1. Fork the repository
2. Create a branch from `main`
3. Make your changes
4. Run `yarn lint` and `yarn format` to ensure code style consistency
5. Run `yarn test --run` to make sure tests pass
6. Submit a pull request against `main`

## Development Setup

See the [README](README.md) for instructions on setting up the project locally.
