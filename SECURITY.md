# Security Policy

## Supported Versions

| Version | Supported |
|---------|-----------|
| 1.0.x   | Yes       |

## Reporting a Vulnerability

Please **do not** open a public issue for security vulnerabilities.

Instead, email [really@awweso.me](mailto:really@awweso.me) with details of the vulnerability. Include steps to reproduce if possible.

You can expect a best-effort acknowledgment within a few days.
